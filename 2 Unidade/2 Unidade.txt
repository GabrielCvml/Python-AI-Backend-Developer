Variáveis e constantes

Variáveis = base de armazenamento que podem mudar com base no tempo.
constantes = base de armazenamento que não podem mudar com base no tempo. 
Para usar constantes em python se nomes de variaveis como letras maiusculas ex: IDADE = 1

Boas praticas
- O padrão deve ser snake cade = nome_da_pessoa;
- Escolha nomes sugestivos.
- Nome de constantes todo em maiusculo
